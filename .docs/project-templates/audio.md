## Script Format
- Include timing estimates for each segment
- Mark music/sound cues with [MUSIC: description] or [SFX: description]
- Mark transitions with [TRANSITION: type]
- Include intro hook (first 60 seconds)
- Mark ad breaks or sponsor segments with [AD BREAK]
- Include outro with call-to-action

## Show Notes Format
- Episode title and number
- Brief summary (2-3 sentences)
- Key topics covered with timestamps
- Links and resources mentioned
- Guest bio (if applicable)
- Transcript link (if available)

## Production Workflow
1. Topic research and outline
2. Script writing
3. Script review (accuracy, flow, timing)
4. Recording preparation (guest coordination, equipment check)
5. Recording
6. Post-production notes (edit points, corrections)
7. Show notes and metadata
8. Distribution and promotion

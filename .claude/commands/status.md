# Status

Get a comprehensive status report of the workspace or current project.

## Instructions

1. **Workspace-level status**:
   - List all projects in `repos/` with brief descriptions
   - Show which projects have uncommitted changes
   - Show which projects have unpushed commits

2. **Active project status** (if a project is active):
   - Git status (branch, uncommitted changes, ahead/behind remote)
   - Recent commit history (last 5 commits)
   - Any TODO items or in-progress tasks
   - Current task list status (if tasks exist)

3. **Session status**:
   - What tasks were planned for this session
   - What's been completed
   - What's in progress
   - What's blocked

4. Present as a concise summary, using tables where appropriate
